# Sense Priority Roadmap

## Phase 1 — Foundation
- Initialize repository
- Add README, .gitignore and project documentation
- Define free tools for MVP
- Prepare backlog and architecture

## Phase 2 — Frontend Setup
- Create React + TypeScript app with Vite
- Configure Tailwind CSS
- Add React Router, Zustand and TanStack Query
- Create folder structure

## Phase 3 — Design System
- Build premium layout
- Create interactive background
- Create reusable cards, panels and tags
- Define colors, spacing and motion rules

## Phase 4 — Core Product
- Unified inbox for email and WhatsApp
- Ticket details page
- Priority rules page
- SLA monitor page
- Real-time dashboard

## Phase 5 — Automation
- Email intake workflow in n8n
- WhatsApp webhook workflow
- Priority score engine
- Alerts and escalation rules
- Error workflow and retries

## Phase 6 — Validation
- Manual integration tests
- Twilio WhatsApp Sandbox tests
- Email tests with IMAP/SMTP
- Deploy frontend on Vercel
