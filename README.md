# Sense Priority

Sense Priority is a smart priority inbox built to receive, classify and organize incoming requests from email and WhatsApp.

## Vision
Create a premium operational platform with an advanced React interface, interactive background, automation workflows and real-time prioritization.

## Core goals
- Unify requests from email and WhatsApp
- Apply priority score rules automatically
- Visualize SLAs and queues in real time
- Offer a premium dashboard experience
- Use free tools for MVP validation

## Free stack for MVP
- Frontend: React + TypeScript + Vite
- UI: Tailwind CSS + Motion
- State/Data: Zustand + TanStack Query
- Database: Supabase Free
- Automation: n8n self-hosted or trial
- WhatsApp tests: Twilio WhatsApp Sandbox
- Email tests: Gmail / IMAP + SMTP
- Deploy: Vercel Hobby

## Roadmap
